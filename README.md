# Social Networking Website

A full-stack social networking website built with **HTML, CSS, JavaScript, Bootstrap, PHP, and MySQL**.  
This project allows users to register, log in, create profiles, and interact through comments.

---

## 🚀 Features
- **User Authentication**
  - Secure registration and login system
  - Password hashing for security

- **User Profiles**
  - Each user has a personal profile page
  - Profile information stored in MySQL database

- **Comments System**
  - Users can post comments
  - Comments are stored and displayed dynamically

- **Responsive Design**
  - Built with Bootstrap for mobile-friendly layouts
  - Clean and modern UI

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Bootstrap  
- **Backend:** PHP  
- **Database:** MySQL
